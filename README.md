# Masterclass de Rust

## Introdução

Rust é uma linguagem de programação **compilada**, de **tipagem estática forte**, projetada para oferecer segurança de memória, desempenho e confiabilidade.

### Instalação

- 🌐 Site oficial: https://rust-lang.org
- 📥 Download do Rust: https://rust-lang.org/tools/install/
- ▶️ Executar exemplos online (Rust Playground): https://play.rust-lang.org

# Motivação

Rust foi projetada para o desenvolvimento de sistemas modernos que exigem:

- Segurança de memória sem *Garbage Collector*
- Alto desempenho
- Concorrência segura
- Alta escalabilidade
- Código confiável
- Open Source
- Comunidade ativa

---

# Áreas de atuação

Rust pode ser utilizado em praticamente qualquer tipo de software.

Entre as principais áreas estão:

- 🔒 Sistemas seguros
- 🛡️ Segurança da informação
- 🎮 Desenvolvimento de jogos
- 🗄️ Bancos de dados

- ⚡ Sistemas embarcados
- ☁️ Cloud Computing
- 🤖 Inteligência Artificial

- 🚀 Centenas de outras aplicações

# Objetivo da Masterclass

Ao término desta Masterclass o participante será capaz de:

- compreender os fundamentos da linguagem Rust;
- desenvolver pequenos programas utilizando boas práticas;
- entender como funciona o gerenciamento de memória da linguagem;
- utilizar bibliotecas externas;
- iniciar projetos próprios utilizando Cargo;
- dar os primeiros passos no desenvolvimento de aplicações Web utilizando Rust.
